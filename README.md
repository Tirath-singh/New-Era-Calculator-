## 📖 Overview

Smart Calculator is a beautiful web application designed with a premium dark theme and smooth animations. It supports multiple mathematical operations while maintaining a clean and user-friendly interface.

This project is perfect for beginners and intermediate developers who want to understand JavaScript DOM manipulation, responsive layouts, and interactive UI design.

---

## ✨ Features

✅ Addition

✅ Subtraction

✅ Multiplication

✅ Division

✅ Percentage Calculation

✅ Power (xⁿ)

✅ Square (x²)

✅ Square Root (√x)

✅ Responsive Design

✅ Live Calculation Display

✅ Calculation History

✅ Error Handling

✅ Keyboard Support (Press Enter)

✅ Modern Animated UI
